## üöÄ Getting Started

All games are designed to be played directly in a modern web browser.

### Playing a Game

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git)
    ```
2.  **Navigate to a Game Directory:**
    Each game resides in its own folder (e.g., `pong/`, `tictactoe/`).
    ```bash
    cd YOUR_REPOSITORY_NAME/GAME_NAME
    ```
3.  **Open the Index File:**
    Locate the `index.html` file within the game's directory and open it with your web browser.

> **Tip:** You can often view a live demo of a game by using **GitHub Pages** for this repository. Check the "About" section or the repository settings for the live deployment link!
